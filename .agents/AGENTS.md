# Regras do Projeto IAMED

- **Preferência de Navegador:** Sempre utilize o navegador Brave para qualquer comando, script, ou automação que envolva abrir páginas web ou interagir com o navegador no sistema do usuário.
- **Execução Local:** Sempre certifique-se de que os comandos de execução iniciam o projeto no Brave usando `start brave <URL>` (no Windows).
